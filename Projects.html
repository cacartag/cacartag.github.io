Pascal Compiler For this project I was tasked with creating a pascal compiler using C. The pascal compiler would be able to first parse the pascal program for all tokens, and define what kind of tokens they were for example if they were numbers, reserved words, or possible variables. Then the program would pass through a syntax, and semantic analyzer, which was created using a decorated parsing tree. Any errors found in the analyzis would be reported to the programmer.
  • C
  • Python
  • Learned the basic concepts of compiler design, which included the creation and implementation of parsing trees given a grammar.
  • Learned how to use scripting languages in order to save time when writing repetive code in compiled languages. In this case Python to write C.
